# WordCounteris – Word & Character Counter

An Android app that counts words or characters in user-provided text.

## Features
- Count words or characters via dropdown selector (Spinner)
- Input validation with Toast warning on empty input
- Modular design – counting logic separated into `TextCounter` utility class

## Tech Stack
- Java
- Android SDK
- Gradle (Kotlin DSL)

## How to run
1. Clone the repo
2. Open in Android Studio
3. Run on emulator or physical device
